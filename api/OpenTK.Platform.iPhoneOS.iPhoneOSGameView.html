ResizeFrameBuffer()

Resizes the frame buffer. On IOS, when the device is rotated from portrait to landscape, if the framebuffer is not recreated then the EAGLContext will stretch the render buffer to fit the screen. If you care at all about aspect ratio, then this behavior is not desired, and in that case this method should be called when rotating the device. See here: http://stackoverflow.com/questions/20326947/opengl-what-need-to-reconfig-when-rotate-screen And also here: https://gamedev.stackexchange.com/questions/75965/how-do-i-reconfigure-my-gles-frame-buffer-after-a-rotation
